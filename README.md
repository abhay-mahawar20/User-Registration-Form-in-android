"# User-Registration-Form-in-android" 
User Registration App

A simple Android application for user registration. This app allows users to enter their personal details and display the submitted data on the screen.

Features
User Registration Form
Enter Name
Enter Address
Select Gender
Select Date
Submit Button
Display User Data in TextView
Technologies Used
Android Studio
Java 
XML Layouts
App Functionality
User enters name and address.
User selects gender.
User chooses a date.
On clicking the submit button, all entered details are displayed in a TextView.
UI Components Used
EditText
RadioButton / RadioGroup
DatePicker
Button
TextView
Purpose

This project is useful for beginners to learn:

Android UI Design
Handling User Input
Button Click Events
Displaying Data in TextView
